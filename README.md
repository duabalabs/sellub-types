# @duabalabs/sellub-types

Shared TypeScript types and DTOs for the Sellub commerce platform. Consumed by sellub-client, sellub-react, sellub-react-native, and every sellub-vendure-* plugin.

> Part of the [Sellub](https://github.com/duabalabs/sellub) commerce platform.

## Install

```bash
npm install @duabalabs/sellub-types
```

## Status

Early preview — API may change before 1.0.
